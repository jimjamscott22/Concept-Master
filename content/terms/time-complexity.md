Time complexity measures how the runtime of an algorithm grows as the input size increases.

It is commonly expressed using Big-O notation, such as:
- O(1) constant time
- O(n) linear time
- O(log n) logarithmic time

```
// O(n) example
for (int i = 0; i < n; i++) {
    System.out.println(i);
}
```
